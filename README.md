# LeagueIQ

LeagueIQ is a football trivia app for fans of the Big Five European leagues — Premier League, La Liga, Serie A, Bundesliga, and Ligue 1.

Players can test their knowledge across multiple game modes (Quick Play, Daily Challenge, Speed Round, Survival, 1v1, Friend Challenges, and Weekly Tournaments), earn XP, level up, and compete on leaderboards.

Built with **Expo SDK 52** (iOS & Android) and **Next.js 14** (web), backed by **Supabase** for database, auth, real-time, and all server-side game logic via Edge Functions. The app supports email/password and Google OAuth, and includes a freemium monetization model powered by Paystack and Google AdMob.

The question bank currently holds **1,048 questions** across 5 leagues, 7 categories per league, and 3 difficulty tiers — each with a "Did You Know" fact.
